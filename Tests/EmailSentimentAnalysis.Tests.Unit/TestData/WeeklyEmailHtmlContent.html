Woolworths (Retail)

From CIO Summit VIC 2015 Lead Import

Project Background

Improving home delivery is critically important to Woolworths. Woolworths are losing customers and sales as a result of a poor delivery service when ordering online. This project is primarily optimisation of the delivery system for the onsite ordering site. This involves route optimisation system replacement as well as architectural changes across the systems that are currently in place to facilitate this.
